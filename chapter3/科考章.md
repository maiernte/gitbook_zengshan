# 卷三

## 童试章第四十九

**父旺官兴，堪期首选**。官衰父发，亦论抡收。官星持世，得日月动爻相生相合，道选无疑。父爻持世，得日月动爻相生相合者，亦要官星有气，始许成名。

**子兴财发于卦中，难游泮水**。子动克官，财动克父，或持于世爻者皆不宜也。

**世破身空兼墓绝，再奋芸窗**。世爻空破墓绝，或被日月动爻冲克及动而变凶者，不能遂志。

**父旺遭伤，收而复弃**。冲中逢合，见弃复收。父母旺相，若化休囚空破墓冲，若得日月动爻相合者，谓之冲中逢合必获绩取。

**父衰变旺，愈出愈奇**。爻化退神，日长日短。父衰动而化回头生、化进神，文章得意胜于前。皆不宜化退神，化退神者，前篇稍遂，后劲不力，衰化退神及化克化空破墓绝或回头克者，窗下午日长，场中一日短。

如亥月丙戌日占道考得丰之革卦

<a class="anchorjs-link gua" id='占道考' href="#占道考"  data-anchorjs-icon=""></a>

$排卦：{亥月丙戌日}{丰之革}{占道考}$

父母持世化进神，日建作官而生世，文章愈出愈奇，后胜于前，定蒙首选，果取案首。

## 岁科考章第五十

**父宜旺相，最忌休囚；官宜生身，不宜克世**。父与世爻旺相及得日月动爻生扶及动而化吉，全无破绽者，定考超等。若此两爻旺相而逢生上等；旺而无生，相而遇扶，无刑冲克破者次之；有一而遇刑冲克者又次之；世爻受克，教刑不免；世父同受克，劣等无疑。野鹤曰：此云是也，亦要人之通变，观其世父两爻轻重而言之，卦之全美世父两爻有生扶，而六冲变冲反吟伏吟动变克化绝即是破绽。父爻受克而无救，难免涂鸦；世爻受克而有扶，亦业姑许。

觉子曰：此以父爻之旺衰而列等第是也，易之卦得进神考于上等，退而考下等，假使卦无进退之神，何以定之？

如申月乙巳日占岁考得大过之鼎卦

<a class="anchorjs-link gua" id='占岁考' href="#占岁考"  data-anchorjs-icon=""></a>

$排卦：{申月乙巳日}{大过之鼎}{占岁考}$

【断曰】：财动克父，乃不通变之论也。此卦乃亥水父爻持世，申月生之，父旺而身亦旺，巳日冲之，为暗动，未土财爻虽动，喜其生助酉金，酉官生世，名为接续相生，一定篇篇华锦。财生官旺优等。后果取超等补禀。野鹤曰：大凡卦遇 六冲变冲及六爻乱动，父旺世旺，还考平等。倘若父衰受刑克及世爻休囚受克及变凶者，孙山无疑。

飞神无助身无克，荣辱无施。父爻出现，不旺不衰，虽无生扶，亦无克制而世爻不被冲克者，可保平等。

伏藏旺相也遭伤，赏罚不定。

如午月乙卯日占岁考得地天泰卦

<a class="anchorjs-link gua" id='占岁考例二' href="#占岁考例二"  data-anchorjs-icon=""></a>

$排卦：{午月乙卯日}{地天泰}{占岁考例二}$

午火月建为父母，而卦中之巳火伏藏于寅木之下，旺而又遇飞神之生，爻逢六合，只不宜日辰克世，发落之日，承宗师面叱念，字法古笔，姑取三等，后悔之。此巳午之火，非又旺也及字端边，但此卦虽多逢，事却少见，存之为法。

以上总而言之，父与世爻旺相遇生扶为上，略有破绽者次之，刑伤有救者又次之，刑伤无救而轻者又次之。世爻父爻被刑冲克破而重者，劣等必矣。全要通变，须卦之吉凶兼用神衰旺之轻重也。

## 增禀章第五十一

觉子曰：占禀若不得其秘，而望灵者少也，何也？占禀用财，财乃文章之忌神，不知其窍，卜岁焉而兼问禀，财爻若旺，父母受伤，则如休囚如何得禀？余有法也，先占岁科之吉凶，再占一卦而问禀，即其秘也。

禀以财爻为用神，财爻持世或动而生世为吉，与求财章同断。有禀而恐换者，忌兄弟持世，兄爻发动及财逢空破墓绝，动而变凶者，有失也，如官鬼兄爻克世者，轻则草粮，重则道罚，倘卦中三合兄局者，亦主草粮。

## 考道才章第五十二

官爻持世生世，我必业收。父母空破墓绝，他不见用。如世爻受克及空破墓绝，或动而变凶、或子孙持身动于卦中，皆不录之。父母生世持世者吉，休囚空破者凶。

## 发案持榜章第五十三

父为用神，衰者宜生旺之日；旺而静者，逢值冲之日；动而逢值逢合之日；空与入墓，必待冲开；破与旬空，须期填实。余又得验，心有忧惧，若得子孙持世，值子孙之日而发案。心有欢喜，若得官星持世，逢官鬼之日而挂榜。

## 庭试章第五十四

典会场断，官星为用，父爻次之。若持世爻得日月岁五生合者，定蒙道拔。官爻世爻三合，定蒙异宠隆恩。世与官爻有一而逢空墓绝刑冲克害及动而变凶者，不详之兆。岁五官鬼冲无世爻者不测之灾。

## 乡试章第五十五

父旺官兴，姓标虎榜。父官持世而旺相，或临日月、或日月动爻生扶、及动而变生，名登龙虎。

子摇财动，名落孙山。子孙财爻持世，及动于卦中，终须下第。

兄弟乃夺标之恶客。俗传兄弟爻兴难上榜，凡见兄爻持世及动者，皆以为之不吉，殊不知有喜有忌，所喜者卦中财爻同兴，喜其兄动以制财也。卦中父官两动者，又喜兄爻持世，父动以生世也。兄弟持世，若临日月或旺相，再得官父两旺、或官动生父、父动生世，必添科名。兄不能抡元而夺魁也，故谓之兄弟乃夺标之客。觉子曰：卦中官鬼子孙同动，官临旺相遇生扶，子孙弱而被克，再宜父动，最怕兄兴。

日月为制煞之将军。官星父母世爻有一而被动爻克制，或化回头克，若得日月冲制克神，亦许得意，故谓制恶煞将军。又如，世爻旺相而官父两爻有一而伏藏不现，倘得日月冲制飞神，提出伏神者，许高发。又如官父世爻有一而不生者，若得日月生扶亦同此意。

二合无冲，定登甲第。官父世爻合成父局官局，不遇日月冲破者，定许连登，倘若合成兄局，得遇日月生扶，官星亦旺者，亦许得意。唯忌合成子孙局者，不为吉也。合成财局虽不为美。然亦得验于吉者。

如卯月甲申日占会试得艮之益

<a class="anchorjs-link gua" id='占会试' href="#占会试"  data-anchorjs-icon=""></a>

$排卦：{卯月甲申日}{艮之益}{占会试}$

寅木旺官持世，申日冲之，暗动又得日辰会成财局，不唯不克世爻，反来生世，一定高捷，果得及第。

六爻竟发空赴科场。旧系六爻竟发，功名恍惚以难成，此论是理。余见占功名必中者，卦象一成，若非旺父生身，定是旺官持世。功名无成者不是子孙持世，即是子孙财爻发动，或是世爻破空，或是六爻乱动，功名必无成。

出现无情，难遂青云之志。父与官星虽则出现，若不生合世爻，或不持世，或世爻空破，亦属无益。父与官星生合世爻，若又被日月动爻冲坏者，亦作无情之喻。

伏藏有用，终辞白屋之人。官父有一而不上卦，须看伏神，若伏而旺相，或得日月冲去飞神，生起伏神，准登科及第。

太岁作官星，终登黄甲。太岁乃国君，父官临太岁，动而生合世爻，乃人臣面君之象，当主君臣契合，连捷无疑。太岁必要入爻，入爻亦要发动，或持世爻，始如此说。

日月冲克世，且守青灯。卦中官父虽旺而世爻休囚空破墓绝及动而变凶，或日月冲克世爻，须知求名在我，我位既失，何以成名。

卦遇六冲，此去难题雁塔。爻逢六合，这回可夺鳌头。

如午月丙辰日占乡试。得兑卦

<a class="anchorjs-link gua" id='占乡试' href="#占乡试"  data-anchorjs-icon=""></a>

$排卦：{午月丙辰日}{兑为泽}{占乡试}$

旧注：卦逢六冲，冲之即散。余见此卦，虽系六冲，世父皆逢旺地，不敢决之，请再占一卦。

又得临之师卦

<a class="anchorjs-link gua" id='再占乡试' href="#再占乡试"  data-anchorjs-icon=""></a>

$排卦：{午月丙辰日}{临之师}{再占乡试}$

此卦官星持世，虽则不旺，初爻官生父旺，与前卦相合，许之吉兆，果得成名。凡得六冲卦，官父世爻俱旺者，不必再占，竟许必得。如父旺者，须再占一卦，再得吉者，以吉断之，后卦凶者，即以凶推。凡得六合卦，亦要官父世爻得地，有一失陷不遇生扶者，虽六合亦无益矣。

父旺官衰终须下第。父爻旺相官临空破墓绝及动而变凶变克虽有锦乡。文章终遇红纱蔽目。

父衰官旺堪许登科。官爻旺而生世持世，若是父爻不旺，谓之福齐文不齐，反许遂志。

应合日生必资鹗荐。官父两旺而世爻不旺，或得日月扶助，又得应爻动而生合世爻者，必得推荐之力。

动伤日克还守鸡窗。应爻动爻虽来生合世爻，而应爻又被日月冲克者，荐亦无功。

世动化官化克蝶梦堪忧。旧有身官化鬼月扶，连步蟾宫。又曰：身兴变鬼，来试方成。又曰：身动化空用旺，豹变翻成蝶梦。皆以卦身而言，余试总无应验，唯验于世爻，而世爻化鬼亦有两验。卦中父母旺相而世爻化出官星者，若不回头克世乃为变官，今科即中，岂等来科？若休囚被克，动而化出官星者，乃为化鬼，不唯不中，且见危灾。化出鬼爻克世者更凶。

如卯月戌辰日占得离之既济

<a class="anchorjs-link gua" id='占科试' href="#占科试"  data-anchorjs-icon=""></a>

$排卦：{卯月戊辰日}{离之既济}{占科试}$

旧注：身官化鬼，月扶鹏程，连步蟾宫。谓卦身如得官爻临之，必有成望，若得出现发动化出官爻，又得月建生之合之者，亦必聊捷。

觉子日：此卦已为卦身，又是世爻变出子水官星，又得月建生合，正合此说，竟未见其连捷，不唯不能连捷，且难保其长年，其故何也？因变出子水之鬼回头克。后客死于途中。

身兴化旺化生，龙池变化。世爻旺相动而变出官星回头生合世爻者，不作变鬼，乃为变官。

世退官星化退，窗下囊萤。世与官父相旺内有一而化进神者，名姓高悬，占乡试必连捷。独不喜化退神，世化退神，若非临场之阻，必无勇往之心，父化退神学问日颓，文章渐腐，官退化神，皓首无成，当于进退章中参考。

财临白虎持身，幛前枕块。旧财临白虎必丁忧，然亦不唯临虎，凡遇财爻持世，或财动卦中，父母又值休囚空破，及财动化父，父动化财，皆枕块之兆。

乾卦如天亦当官父两旺。雷声如震犹有空破之嫌。古以乾震之卦既以为吉，余试务必官父两旺，始以吉断，否则未见为吉卦。

衰旺克冲，轻重须宜细玩。墓绝空破，应期章内详之。前应期章宜熟记，各门各类，无不问其应期，熟则触类旁通，随心应变。凡遇用神衰要有根，旺宜冲，刑克要有救，冲要逢合。
