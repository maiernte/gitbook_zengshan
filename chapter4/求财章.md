# 卷四

## 求财章第六十八

诸书之论求财，弗如黄金策里真论，余屡试验，惜乎未经分别，今以前段为总论，以后贤易晓，内有屡试而不验者，删之，余另得验者，增之。

财旺福兴，公私称意。财空福绝，上下连心。上至国计，下至营谋，无不以财为用。公私占卜皆以财为用神，子孙者乃生助财爻之元神也，俱宜旺而化吉，又宜生合世爻，公私皆得如意。若逢衰墓绝空刑冲克害，动而变凶，日月冲克破者，上下皆连心矣。

有福无财，兄弟交重偏有望。兄弟乃劫财之神，占财之忌也，财与兄爻同动，必主阻隔破耗，倘若得子孙与兄弟同动，又为可喜，兄动生子，故谓之有望。财爻兄爻子孙爻俱动于卦中者，更为可喜，其故何也？兄动生子，子动生财，其利甚厚且有久远源流长之利益也。

兄与财振，官爻发动亦堪求。兄爻与财爻同动而官鬼亦动者，亦许得财，以官鬼克制兄爻之故耳。

财福俱无，莫若守株待兔。财与子孙爻俱不现于卦中，或现于卦中又值休囚空破墓绝，或被刑冲克害，求则无益，不如守株待兔。觉子曰：倘得日月为财，或伏而旺者，亦可求也。

父兄爻动，无殊缘木求鱼。父动克福神，兄动克财爻，此二爻动于卦中，凡有所图，水中捞月。

如酉月戊午日占求财得革卦

<a class="anchorjs-link gua" id='占求财' href="#占求财"  data-anchorjs-icon=""></a>

$排卦：{酉月戊午日}{泽水革}{占求财}$

【断曰】：卦中财爻不现，亥水兄爻持世，父临月建生助兄爻，如缘木以求鱼也。

多财反覆，必须墓库以收藏。卦中财临日月，谓之重叠，若许求谋得意之时，须待财爻入墓之日。且如水作财神辰日而得，余仿此。

无鬼分争，又怕友重而阻滞。兄弟乃争夺阻隔耗财之神，宜官鬼动而制之，以免分争之患。觉子曰：兄爻发动，喜鬼动以制之，倘卦中兄爻安静者，又不宜乎鬼动，反曳财爻之气也。且有口舌。

兄如太过，反不克财。旧注卦中一位兄爻动者，最为利害，如若兄弟爻多动者，反不劫财。觉子曰：非也，兄爻多者，待兄爻入墓之日，及克损爻之日，必劫其财，谓之太旺者损之斯成。

世遇兄临，必难求望。古以卦身临兄爻，难于求望，余因卦身不验，以兄爻持世，难遂图谋。野鹤曰：全在人立通变，兄弟而持世者，虽则为忌亦有不当忌者。

如未月丁卯日借贷得晋卦

<a class="anchorjs-link gua" id='占借贷' href="#占借贷"  data-anchorjs-icon=""></a>

$排卦：{未月丁卯日}{火地晋}{占借贷}$

【断曰】：虽则兄爻持世无财，但喜卯日即是财星，古以财爻持世冲世者必得，况应爻未土旺而生世，明日必获，果得财，应辰日者，世动逢合之日也。

又如巳月丁巳日占求财得即济变涣卦

<a class="anchorjs-link gua" id='占求财例二' href="#占求财例二"  data-anchorjs-icon=""></a>

$排卦：{巳月丁巳日}{既济之涣}{占求财例二}$

【断曰】：若逢久远之财有否，则无财。若问目下之财，明日戊午必得，其故何也？兄临世爻日破月破，不克变出之财，日月俱作财来冲世，只因应爻而逢空，明日冲实，定送财来，果于次日送来。

财来就我终须易，我去寻财必是难。旧注凡遇财爻生世合世克世持世，皆谓之财来就我，若卦中虽有财爻，不来生合世爻亦非我之财也。

野鹤曰：此说财不生合世爻，非我之财，即如前二卦兄爻持世，何当财来生合，总要断卦之灵机应变，彻悟其理，自然触类旁通，不可拘执。

福变财生利源滚滚。财得子动生之，如财之有源也，其利丰厚，倘得子动化财，财动化子，皆同此断。

兄伤鬼克，口舌纷纷。财爻持世，兄爻亦动者，不独劫财，还防伤已，如若鬼动克世者更凶。

觉子曰：黄金策云，兄连鬼克，口舌难逃，谓兄动变鬼冲克世爻者，不独无财，还防口舌，殊不知兄动变鬼，即是兄化回头之克，自顾不暇又焉能冲克世爻耶？

财局合福德，万倍利源可追。世与财爻或与子孙爻三合成局，诸占物追财丰。

野鹤曰：须要在局中或是合成子孙财局生世，方为全美，倘如合成父局，劳碌卒苦，合成兄局破耗多端，合成鬼局，口舌灾非，克世者更凶。

岁君逢劫杀，一年生意无聊。劫杀者乃兄弟之爻也，兄临太岁一年破耗。

## 谒贵求财章第六十九

世克官，官生世，须宜谒见；财临破，鬼克世，枉费奔驰。野鹤曰：见贵有两间也，为名者用官，为利者用财，皆宜持世及生合世爻为吉，如逢空破，墓绝及动而变凶者，徒劳奔驰之力耳。倘若世爻化鬼化克及随墓助伤者，更不宜行，若非到彼之凶，定在中途遇祸，凡谒贵者同断。世动遇空，身兴遇破。世动化空破者，勿曰不吉，乃应到彼之月日也。

如申月丁卯日占出行谒见贵人得同人

<a class="anchorjs-link gua" id='占出行谒见贵人' href="#占出行谒见贵人"  data-anchorjs-icon=""></a>

$排卦：{申月丁卯日}{天火同人}{占出行谒见贵人}$

【断曰】：官星持世而空，出亥日得见，官府财利如心，果于出空之日见官，财利得意，月建财爻生世之故耳，官世相合，财动相生，财持世，官无缺陷，再得日月照临，彼此缘投有幸。日月照临者，日月生世生财，或世爻财爻而临日月是也。

为名宜父动，因利忌兄兴。凡与谒贵既得世与官星相合相生，必然得见，若为名利，须宜另占，不可此卦而断之，为名者宜父官两旺，为利者宜财福两旺。

六合六冲须看用，反吟化退枉奔驰。爻逢六合亦要用神旺相，世与财官相合乃为吉兆，卦逢六冲必须用神受克失陷始作凶征。卦得反吟及世动化退多见难于行动，勉强而去，空返而归。

贵人谒贵宜世应以相生，平人见贵宜官贵而相合。贵人谒贵人，宜于世应相合相生，平人见贵人，宜官贵相合相生及官星持世。

觉子曰：有心而问见贵求名者，官星持世生世必得官。有心问谒贵者，官星生世合世而得见，切勿以此得见贵人之官星换作成名而断也。

贵人谒贵，世应若相冲克，财伏而空，去之何益？

平人见贵，官克世世变鬼变克更凶。平人见贵，官克世变鬼变克勿去，去则须防不测，财官生世合世为吉。凡出行求财谒贵出行章参看。

## 为贵人奔走效力财章第七十

最重财官两旺，不宜世应皆空。世遇财应临福，我益他损；应临官世逢破，他益我损。兄爻持世，不如闲处安身；应克世爻，防虑他人反目。世遇财官临应，无不遂也。世逢空破墓及动而变凶，皆非吉也。应克世兄克世，见灾还浅。鬼克世世化鬼，得病不轻。

## 开行开店及各色铺面章第七十一

世为已，应为人，大宜相合。财为本，福为基，最喜同兴。古法应为伙计，又为下顾之客。黄金策曰：开行定主有人投。觉子曰：须在来人之念；有心而问伙计，应为伙计，若得世应相生相合，彼此同心。应生世，他益于我。世生应，我益于他。相克相冲而情有变，应爻克世，我被他瞒；世克应爻，他从我愿；应临元武兄爻，暗中盗骗；应临官鬼克世，明降灾非；世财应兄，受他之累；世财应子，得他之力。世应空合，彼此有虚诈之心也。来人之念重在买卖者，须以应爻为下顾之客，应爻生合于世，生客来交成故旧；世应相冲相克，熟人终久变成仇。

鬼作灾非须忌动，财为活计畏刑冲。鬼动招非克世最忌，古云：鬼现爻中见非，日有鬼摇克世灾祸连系，须看鬼值何神克世，须于家室章内详之。财为资本，忌衰墓空破及动而变凶，或日月刑冲克，更忌世与财空，俱开不成。

鬼兄发动有制何妨，随墓助伤多凶少吉。鬼动招非，若得子孙制鬼，或日月冲克鬼爻，谓之有制。觉子曰：凡遇兄鬼同动者，不可曰鬼动招非，以鬼能制兄爻，不劫财也。兄弟又为阴隔破耗之小人，得此鬼动可以制服，独不宜乃为助鬼以伤身也。

卦得反吟多反复。卦得反吟，欲行不行，行而又止。才开而又剪，剪而又开，倘店开久矣，遇此一定更迁。

冲中变合再重兴。爻逢六合及六冲卦变六合或世财爻子孙共成三合，定是前后重兴，惟六合变冲则不宜也，眼前热闹，终久萧条。

合伙不嫌兄弟，蚀本内外无财。世应临兄不吉，兄爻发动不吉，倘若世临兄爻，日月作财星冲克世者，反为吉也。内外无财伏而又空，必然之蚀本，屡验。

## 投行损益章第七十二

官若兴隆，行主可千金之托；应如空破，牙人无毫忽之能；官旺应旺，生合世爻，其人可托，如静衰空破，即是奸险小人。倘若交财，定遭欺骗。世被应伤，遭他陷害，财逢兄克，虑彼相瞒；应克世爻，帐目难清，兄克财爻已入套矣。兄动货难脱卸，子兴物宜交共；兄雀并摇，难逃口舌，武鬼同发，须虑盗贼。

## 囤货卖货章第七十三

财爻衰者须停脱，妻旺当时可脱之；积货卖货皆不宜财逢空破及动而变克；财衰宜囤，财旺宜脱；如春天占得财爻属木，谓之旺，宜速发卖；若临金水土者，宜堆积也。临火为旺，亦不宜堆；此论四时之衰旺也。觉子曰：衰虽可买，须查其根底，虽衰宜有根，将来再发，若无根底，终久难于出脱，不可不知。

如寅月己亥日占收货得颐卦。

<a class="anchorjs-link gua" id='占收货' href="#占收货"  data-anchorjs-icon=""></a>

$排卦：{寅月己亥日}{山雷颐}{占收货}$

古以财爻衰者宜收货，此卦辰戌之财衰而不遇生扶，谓之无根，将来终不得价，不可收买，但求问者不听，后来存货堆积四年，朽坏大半，大亏其本。

货有定时，须凭时令，物无定者，指利而占。定时者有贵贱之时也，比之麻葛得价于夏令，棉花得价于冬天，又如木料绸缎无定时也。

财化进神不可收积，化退者，其价将落，有货速宜出脱。

兄爻持世，破败之端；觉子曰：算甚命，问甚卜，亏人是祸，财将义取者，神告显然。财之不义者，神不告矣。

财值旺衰墓绝，自有得令之秋，妻逢冲破伏空，岂无乘除之日。财爻太旺旺者得利于墓库之月，休囚者出脱于生旺之时，入墓必待冲开，逢绝当逢生旺，月破者填实之，被克者冲去克神之秋，伏藏者得价于出现之时，旬空者冲空实空之月，如逢合住亦待冲开。

如子月己丑日占何时得价得归妹之兑

<a class="anchorjs-link gua" id='占何时得价' href="#占何时得价"  data-anchorjs-icon=""></a>

$排卦：{子月己丑日}{归妹之兑}{占何时得价}$

【断曰】：卯木财爻子月生之，财有根蒂，目下不得价者，被申金动而克制，待寅月冲去申金，其价必涨，果于正月勃然大长，此乃财爻被克待冲去克神之月。

财逢合而当令。遇合者有应动而逢合之月而得价也。亦有财爻衰静，占时得遇日月生合者，其价目下正及时也。

如寅月己酉日占脱货于何时得贲卦

<a class="anchorjs-link gua" id='占脱货于何时' href="#占脱货于何时"  data-anchorjs-icon=""></a>

$排卦：{寅月己酉日}{山火贲}{占脱货于何时}$

【断曰】：亥水财爻，酉日生之，寅月合之，目下正及时也。但因世爻暗动，身动方可出脱。彼曰：我闻某处得价，欲移货去卖，不知可否？余曰：卦已明现，速宜行之，果去，不数日，得价出脱。此乃爻逢六合，财逢月合，及时脱货也。

财逢冲而当起；财旺而静或旺而空得今日之日神冲者，其价渐渐涨矣；倘若静而休囚，日冲为破，勿以此断。

## 卖货宜守宜动章第七十四

内财衰而外财旺，往他乡。外卦财旺生合世爻者，持世者，宜往他乡；倘若财不生合世爻及世动化凶者，他乡花如锦，我去不逢春。

应财外而世财内，须查动静；应爻以外财爻持世者，最好收财，必须待应爻发动而与世爻相生相合，不然的话，应财再旺，我也收不到好处。

六冲须宜别图，六合还宜坐守。卦得六冲，虽宜改图，余屡试之财旺者何须更改，财衰者改亦无益。野鹤曰，凡得此象，须令本人再占，占问更改何如？卦得财世两旺，许之改图，再逢衰者还须耐守。卦得六合，亦看财爻衰旺，财持世而动者，不可坐守，衰而静者，守待旺时可也。

财衰变旺，先屈后生，财旺变衰，先金后土。谓财爻动而变也，财衰变旺，目下虽贱，将来必长。财旺而变衰绝，如若卖货宜于急卖，倘若收货，切不可收。买货与卖货相反，财爻旺而可卖，衰者宜买，内卦财衰，宜于外卖，世爻财旺，近地相宜，财化进神，宜往外方发卖，世爻化退，宜回近地出脱。

## 往何方买卖章第七十五

屡次试验都须指明方向才能占卜。

如午月戊子日占往楚买豆得临卦

<a class="anchorjs-link gua" id='占往楚买豆' href="#占往楚买豆"  data-anchorjs-icon=""></a>

$排卦：{午月戊子日}{地泽临}{占往楚买豆}$

断：亥水财爻休囚于外，所以楚地豆子必贱，子水日建而生世爻，必能得厚利。但闲间爻兄弟月生日合，耗费的钱一定多。后果中间人作弊，告到官府，误了一个月。幸好到地头上价格上涨，果然得了厚利。此乃指其地而占者无不验也。

## 买何货为吉章第七十六

古以金财而像珠宝玉石，水财以类鱼盐，火财陶冶东品，五谷为土财，青龙水利又兼喜庆之用，白虎金财，丧仪屠宰之物，玄武水利，朱雀火宜，蛇利于出入，勾陈利于农工。

如未月戊申日占往粤东探亲可带何货归来得利得旅卦

<a class="anchorjs-link gua" id='带何货归来得利' href="#带何货归来得利"  data-anchorjs-icon=""></a>

$排卦：{未月戊申日}{火山旅}{带何货归来得利}$

【断曰】：爻逢六合，世应相生，此行遂意，如问置实何货，余无此法，不敢相误，但喜既得吉卦，任尔冬天买扇，夏天买毡亦得厚利。后此人买攒香翠毛而归，俱得大利。若执古法断者，此卦酉金为财可买珠宝玉石耶？财临玄武，可买水利鱼盐耶？

## 借贷章第七十七

世逢兄，何须开口，应空破，难遂我心，财若空破，休指望。子兴财发可相求，世应两爻皆不宜月破旬空，兄弟一爻切不可临身及动。

如未月丁卯日占借贷得兑之震

<a class="anchorjs-link gua" id='占借贷例二' href="#占借贷例二"  data-anchorjs-icon=""></a>

$排卦：{未月丁卯日}{兑之震}{占借贷例二}$

【断曰】：兑卦属金变震卦属木，金克木为财，卯木财爻化退神酉金兄动亦化退神，幸得财临日辰旺而不退，明日辰日合住酉爻必得，果于辰日。

## 放债索债章第七十八

兄临世，放则无归。兄弟爻持世，索财不予。应鬼克身，防负义。应财生世，定怀忠。古以世应同人，放债必然连本折，谓不独世临兄弟忌之，应临兄者亦忌。官鬼克身，美意终成恶意。

## 买卖六畜章第七十九

山野兽禽，须寻福德。六畜牛犬，亦看子孙。不拘家禽野兽皆以子孙为用神，末看财爻持世，子动生之，或日月动爻生财或财动化子孙，乃吉象也。不拘买卖畜禽，多多益善，大有利息。子孙持世，财爻合世，亦有利益。独防鬼变子孙，子孙变鬼，父化子，子化父，父兄持世，财临空破，不可行之。

如丑月丁卯日买马往南发卖得鼎之大有

$排卦：{丑月丁卯日}{鼎之大有}{买马往南发卖}$

【断曰】：内卦子孙虽临月建，不宜动而变鬼，外卦未土子孙，又临月破，不宜买，幸而间爻酉财暗动生世，马虽多死，财不致于大耗，后果买马七十余匹，一路多死，仅存二十八匹，幸而得价不然大亏其本。

## 博戏章第八十

有技力，首重世应，若胜以得财者，兼用财。又世喜临于日月，财宜日月生扶，世爻动而化吉，应带财而生世，皆我胜。世财休囚空破或被刑冲克害，及卦中父动者，多破其财。

$排卦：{巳月戊申日}{渐之巽}{占鹌鹑}$

【断曰】：子孙持世以临日辰，父爻虽动，亥水制之，独不宜亥为月破，虽可制火，今日巳午时切不可斗。彼曰：何也？余曰：巳午时午火父母得令，必克子孙，果于早胜，午时又大败破财。

## 请会摇会章第八十一

请会须宜财旺，更宜世应相生。卦遇六冲，财虽旺而不久。六冲变合，世应相克亦成。六合变冲，有始无终。财逢绝破，速宜止之。

摇会得否，旺财持世必得。如占何时可得，财破财空填实之日，财若伏藏，出现之日。财太旺者，逢墓之日。财衰绝者，逢相生之日。倘得财化回头之克及衰静而逢空破又土遇六冲者，会场必散，终无得会之日矣。

## 行险求财章第八十二

觉子曰：古以青龙财爻为吉，余以此非营生之本，乃丧心之事也。行此丧心之事，亦敢告之于神耶？既告之于神而神若以吉凶而报者，乃神之教人而作丧心之事也，可谓之大道之易卦耶？可谓之伏羲之大神耶？然不独此，凡占非礼非义之事，不忠不孝之谋，一念之恶，神随知之，即不告之于神，亦遭天谴。书曰：惠迪吉，从逆凶，何须问卜？
